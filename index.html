Problem

Formal discovery stalls when (1) search spaces explode, (2) results aren’t replayable, and (3) teams operate blind without telemetry.

  • Combinatorial proof search; heuristics drift across machines and versions.
  • Non-determinism blocks reproducibility, peer review, and audit trails.
  • Minimal observability → slow iteration, weak anomaly response.

Solution

ProofX unifies multi-prover orchestration with deterministic replay and entropy-aware scheduling, streamed into operator dashboards that keep proofs honest and search productive.

Determinism

Seeds, routes, and solver portfolios persisted per run; identical replay across machines and time.

Observability

Live traces, entropy/decay overlays, anomaly flags, run diffs; SRE-style insight for symbolic systems.

Composability

FOL, SMT, finite models, SAT bridges — swappable, typed, and logged with proof obligations.

Architecture

Deterministic solvers stacked with symbolic search and first-class observability. Every attempt becomes a data point in a growing evaluation wall.

  • Solvers: SMT, FOL, finite model testing, SAT bridges
  • Search: entropy-aware scheduling, curriculum runs, counterexample harvesting
  • Telemetry: traces, dashboards, anomaly flags, decay maps

Truth-aligned latency: fastest path from conjecture to verifiable evidence.
